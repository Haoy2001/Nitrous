# Nitrous
### A 3D driving game made using Processing 5 at Electric City Hacks 2018
#### Features
 - Realistic driving physics
     - Destructible terrain
     - Accerator and brake
     - Increased drag when not driving on roads
 - Randomly generated map using Perlin Noise
 - Custom 3d models
